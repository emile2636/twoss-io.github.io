OpenStack Taiwan User Group

hhttps://www.facebook.com/groups/openstack.tw/

OpenStack Taiwan User Group 是由台灣 OpenStack 使用者與開發者們共同創立的公開性社團，並利用 Meetup 定期在各台灣舉辦技術與推廣分享活動。而近年來由於其他開放式原始碼專案，如 Ceph, Kubernetes 與 SDN 等紛紛與 OpenStack 進行密切整合，故本社團也加入 Cloud Native Taiwan User Group 以邀請這些領域專業講者來探討上述專案在 OpenStack 中是如何運作。目的是藉此群策群力來使該社團快速成長，以對台灣在雲端開源平台的貢獻，另外我們更希望該社群持有 Cloud Native 的概念，並將此推廣至台灣企業、學校、政府單位等。

SITCON 學生計算機年會

sitcon.org/

SITCON 學生計算機年會係一學生自發組成的研討會，秉持著以學生為主軸的核心價值，藉以凝聚與傳遞學生的力量。希望藉由提供一個經驗交流與資訊技術實務分享的平台，使已有成就者能夠傳承己身經驗，嶄露頭角者在激盪中獲得靈感與啟發，並同時讓初入茅廬者得以對資訊科學有更深入的了解；進而達到「學以致用、教學相長」的目標。

Ubuntu-TW | Ubuntu 台灣社群

www.ubuntu-tw.org/

Ubuntu 台灣社群，簡稱 Ubuntu-TW，成立於 2006 年，是全球 Ubuntu 在地社群中相當早成立的在地社群之一。Ubuntu-TW 由多位義工組成，旨在推廣Ubuntu這個由 Linux 核心及多個自由/開源軟體組成的作業系統(或通稱「發行版」)，協助解決使用者在安裝、使用 Ubuntu 時所遭遇到的疑難雜症，以及 Ubuntu 的在地化，並與其他社群合作，推廣自由軟體/開源軟體及開放文化。

開源應用技術研發人才培育計畫-工研院資通所

www.itri.org.tw/eng/

行政院推動「數位國家‧創新經濟發展方案」 (2017~2025年)，簡稱DIGI+，做為鞏固「五加二」產業的基盤，推動重心從硬體移往軟體等應用面，包括了數位經濟、共享經濟、電子商務及金融科技等。 工研院資通所與資策會以DIGI+發展方案，推動主軸「跨域數位人才」之推動策略「鏈結國際開放創新資源，加速設計開發能力」推動策略為政策依據，預計以開源技術支援5+2產業創新，先以培育開源技術應用菁英人才為基礎，深化開源軟體技術能力，並且配合利用開源技術的研發成果，培養工程師的開放程式客製化與整合能力，成為國際開源領導者。 本分項計畫針對開源大數據人才培育，結合產政學與社群能量，積極運用開源共享資源，培養菁英平台開發服務人才，擴展開源文化與產業效益。

ITRI OpenStack Distribution (IOD) 雲端資料中心解決方案

工研院研發的IOD採用OpenStack架構，可滿足營運商等級需求的大型雲端資料中心IaaS解決方案，且適用於企業建置私有雲、公有雲和混合雲。 IOD整合高效能分散式區塊儲存DISCO for Cinder plug-in、動態網路規劃Peregrine for Neutron plug-in、實體資料中心管理技術PDCM(Physical Data Center Management) ，WEB應用防火牆WAF(Web Application Firewall)，HA強化平台高可用性，具備實體主機及虛擬資源之監控管理，確保營運服務品質。此外，另於IOD上加裝OpenStack Ceilometer與Trove專案，開發雲系統監控管理與雲資源量測分析、資料庫管理與日誌監控兩項技術。於Ceilometer專案上，著重於資源收集機制以及數據儲存技術的研究。於Trove專案上，定義出更好的日誌管理機制，提供雲資料庫的除錯及備份技術，方便管理者使用。

這是一個專業服務、以人為本的時代，晟鑫科技的經營理念由傾聽客戶心聲開始，充分發揮團隊合作的精神，集思廣益從客戶角度多面向的思考各項需求及其必要性，提供專業、客觀、全方位的各項服務與諮詢，講求的是與客戶間融洽的互動與全心信賴，並建立長遠密切的合作關係，以極致完美的表現達到客戶最高滿意度為第一目標。
